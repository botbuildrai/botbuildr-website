# BotBuildr.ai — Editorial Landing

Production implementation of **BotBuildr Landing - Editorial.html** from the Claude Design handoff bundle.

## Quick start

```bash
npm install
npm run build
npm run preview
```

Open the URL shown in the terminal (default `http://localhost:4173`).

> **Note:** `npm run dev` may fail when the project lives under a path containing `#` (e.g. `Claude Code Project #3`). Use `build` + `preview` instead, or move the folder.

## What's included

- Scroll-scrub ISS hero video (`public/assets/botbuildr-iss-hero.mp4`)
- Editorial hero headline + gold rules
- Proof stats, automation features, how-it-works timeline
- Ship grid, case study, 4-beat journey, final CTA
- Glass nav, reveal animations, count-up stats

## Deploy

```bash
vercel deploy --prod
```
